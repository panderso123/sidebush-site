# SIDEBUSH ART - FACEBOOK VISUAL DESIGN SPECS

## Design System

### Color Palette (Fixed)
- **Primary Dark:** #0a0a0a (near-black background)
- **Accent Gold:** #ffd700 (bright, prominent)
- **Secondary Gold:** #cc9900 (darker for depth)
- **Text Primary:** #ffffff (white)
- **Text Secondary:** #f2f2f2 (off-white)
- **Text Muted:** #9a9a9a (gray for secondary copy)
- **Card BG:** #141414 (dark gray for contrast)

### Typography
- **Primary Font:** Georgia (serif)
- **Fallback:** 'Times New Roman', serif
- **Headline:** 48pt, bold, #ffffff, letter-spacing: 2px
- **Body:** 22pt, regular, #f2f2f2, line-height: 1.6
- **Accent Text:** 18pt, regular, #ffd700, letter-spacing: 1px

### Image Specs
- **Optimal Size:** 1080x1080px (Facebook square format)
- **Alternative:** 1200x628px (cover/link format)
- **Orientation:** Square for product features, landscape for collections
- **Quality:** 300 DPI source images scaled to 72 DPI web (1080px)
- **Effect:** Grayscale overlay with color transition on hover (CSS)

## DESIGN TOOL RECOMMENDATIONS

### Option A: Canva (Easiest)
- Create templates in Canva
- Use brand colors (#0a0a0a, #ffd700)
- Add images from assets folder
- Export as PNG 1080x1080px
- Schedule via Facebook Business Suite

### Option B: Adobe Express
- Similar to Canva
- Better typography controls
- Professional filters available
- 1080x1080px export

### Option C: Manual HTML/CSS
- For full control
- Can create exact specifications above
- Use inline styles for colors/fonts
- Export as static image or live HTML

### Option D: Figma (Most Control)
- Design system approach
- Reusable components
- Exact pixel specifications
- Prototype variations
- Export at multiple sizes

## Accessibility & Mobile

### Mobile Optimization
- All text readable at 400px width (phone size)
- Headline: 32pt minimum on mobile
- Body: 14pt minimum on mobile
- Padding: Increase to 20px on mobile
- CTA button: 44px minimum height (thumb-friendly)

### Contrast Requirements
- Text on image: Use gold (#ffd700) for dark images
- Text on dark bg: White (#ffffff) for readability
- Emoji: Increase visual hierarchy

### Color Blindness
- Don't rely on color alone to convey meaning
- Use emoji + text labels (not just color)
- Example: "🌆 Travel Cityscapes" not just gold text

## A/B Testing

**Test these variables:**
1. Emoji presence (with vs. without)
2. Headline length (short vs. long)
3. Image brightness (bright vs. muted)
4. CTA text ("Shop Now" vs. "View on Etsy" vs. "Discover")
5. Posting time (7am vs. 7pm vs. 2pm)

**Measure:**
- Click-through rate (CTR)
- Engagement rate (likes + comments + shares)
- Save rate (indicates high intent)
- Reach (impressions)
